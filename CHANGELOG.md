# Changelog

## [1.0.0] - 2026-01-07
### Added
- Multi-language support (English, Catalan, Spanish).
- Rich text help window with Markdown-like formatting.
- Mode 1: Create CBZ from image folders with size estimation.
- Mode 2: Organize loose images into series/chapters automatically.
- Mode 3: Convert archives (ZIP/RAR) to CBZ/CBR.
- Theme support (Light/Dark).
- Activity log and summary report.

### Changed
- Refactored project structure for GitHub compliance.
- Optimized UI layout for better scrolling and resizing.
- Improved error handling and logging.
